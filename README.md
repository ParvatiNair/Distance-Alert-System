# Distance-Alert-System
Distance alert system uses Arduino Uno and HC-SR04 ultrasonic sensor to check the distance of the object in front of it and accordingly alerts the user if the object is too close 

Requirements:
1. Arduino Uno 
2. HC-SR04
3. 12V Buzzer
4. 7 White LEDs
5. 7 1kohm resistors
6. USB cable
7. Jumper cables
